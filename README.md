# Пранаяма Таймер - Нади Шодхана
![](/README/nadi-shodhana.png)

PWA-приложение для практики дыхательной техники Нади Шодхана (альтернативное дыхание). Приложение работает оффлайн и может быть установлено на мобильные устройства.

## ✨ Особенности
* 🧘 Таймер для пранаямы - настройка длительности вдоха, задержки и выдоха

* 📱 PWA - устанавливается на телефон как нативное приложение

* 💾 Автосохранение - настройки сохраняются автоматически

* 🎨 Интуитивный интерфейс - красивый и понятный дизайн

* 📊 Визуализация прогресса - прогресс-бар и цветовая индикация фаз

* 🔄 Автодеплой - автоматическая выгрузка на хостинг при обновлении кода

## 🚀 Быстрый старт
Онлайн версия
Откройте [nadi-shodhana](https://vanyukov.github.io/kv.nadi-shodhana/) в браузере и установите как PWA.

Локальная установка
Клонируйте репозиторий:

`bash`
```
git clone https://github.com/vanyukov/kv.nadi-shodhana.git
```
Откройте index.html в браузере или используйте Live Server.

## 📖 Как использовать
Установите параметры дыхания:

* Вдох (секунды)

* Задержка дыхания (секунды)

* Выдох (секунды)

* Количество циклов

Нажмите "Старт" для начала практики

Следуйте указаниям приложения:

🟢 Вдох - зеленый индикатор

🟠 Задержка - оранжевый индикатор

🔵 Выдох - синий индикатор

Приложение автоматически переходит между фазами и циклами

## 🛠 Технологии
* HTML5 - семантическая разметка

* CSS3 - адаптивный дизайн, CSS Grid, Flexbox

* JavaScript - логика таймера, PWA функциональность

* LocalStorage - сохранение настроек

* Service Worker - оффлайн работа

* Web Manifest - установка PWA

## 🔧 Настройка автодеплоя
1. Настройте Secrets в GitHub
Перейдите в Settings → Secrets and variables → Actions и добавьте:

* FTP_SERVER - адрес FTP сервера

* FTP_USERNAME - имя пользователя FTP

* FTP_PASSWORD - пароль FTP

* FTP_DIR - папка на сервере (например /public_html/)

2. При пуше в master ветку
Автоматически запустится деплой на ваш хостинг.

## 📱 Установка как PWA
Android (Chrome):
Откройте приложение в браузере

В меню выберите "Добавить на главный экран"

Подтвердите установку

iOS (Safari):
Откройте приложение в Safari

Нажмите кнопку "Поделиться"

Выберите "На экран «Домой»"

Подтвердите добавление

## 🎯 Рекомендации по практике
Нади Шодхана (Альтернативное дыхание)
Балансирует энергию в правом и левом каналах

Успокаивает нервную систему

Улучшает концентрацию

Стандартные соотношения:
Начинающие: 4-8-8 (вдох-задержка-выдох)

Продвинутые: 4-16-8 или 5-20-10

Количество циклов: 10-30
